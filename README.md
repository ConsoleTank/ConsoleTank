## 总结
 悟性很高，天赋很强，但基础知识薄弱。学校已经学过的内容没有形成知识链，还需要需要多加练习。
 本课程只是随全部掌握，但理解的还是不太到位。也属于正常现象，需要自己在后半学期课余时间多写。
 
 **指针部分**内容需要在好好理解理解这是重中之重。
 
 **面向对象方面**：封装的意识还不够强，还停留在理论阶段，
 
 **建议补充知识**：计算机结构 原理 组成相关的硬件知识。从计算机的工作流程和原理上理解学校所学的知识点。
